# LiteSpeedTest

LiteSpeedTest is a simple tool for batch test ss/ssr/v2ray/trojan server. 

### Usage
```
As proxy:
    lite --link vmess://ABCDEFGHIJKLMNOPQRSTUVWXYZ
    lite --link ssr://ABCDEFGHIJKLMNOPQRSTUVWXYZ

As test tool:
    lite
```

### Build
```bash
    go get -u ./...
    go build -o lite
```
